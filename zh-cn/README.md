Marboo是专业的标记语言文件预览器，支持Markdown、reStructured Text、Org-mode、ePub、Pdf等文件的预览。使用三栏式布局。

左栏是文件夹栏，显示本地文件夹；
中栏是文件栏，显示文件夹中的文件；
右栏是预览栏，显示文件的预览效果。
